# WIFI_MAP

An ESP32 and FillperZero wifi mapping

## Build and flash

$ `idf.py flash`

### Build and flash and monitor

$ `idf.py flash monitor`

## Connection FlipperZero and ESP32

| Flipper Zero GPIO | ESP32 |
| ----------------- | ----- |
|      GND          |  GND  |
|      RX           |  TX0  |
|      3V3          |  3V3  |
